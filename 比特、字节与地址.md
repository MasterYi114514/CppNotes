# 1. 比特与字节

在计算机中，最小的数据单位是比特（bit），它可以表示两种状态：0 和 1。
一个字节（byte）通常由 8 个比特组成，可以表示 256（$2^8$）种不同的状态。

---------------------------------------------------------------------

比特（bit）：  0/1
字节（byte）： 0000 0000 (8个比特，每个位置上都可以是0或1)

---------------------------------------------------------------------

我们用小写的 b 来表示比特，用大写的 B 来表示字节。

```
b = bit = 二进制位
B = byte = 字节 = 8个比特
KB = 千字节 = 1024字节
MB = 百万字节 = 1024KB
GB = 十亿字节 = 1024MB
TB = 万亿字节 = 1024GB
Mbps(Mb/s) = 兆比特每秒
MBps(MB/s) = 兆字节每秒
```

# 2. 内存与地址

内存是计算机中用于存储数据和程序的硬件设备，它由许多字节组成。内存的地址是用来标识每个字节位置的唯一标识符。
