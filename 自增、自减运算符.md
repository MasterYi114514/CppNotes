# 1. 自增运算符

自增运算符是 `++`，它的作用是将一个变量的值增加 1。

自增运算符有两种使用方式：前置自增和后置自增。

## 1.1 前置自增

前置自增的语法是 `++变量名`，它会先将变量的值增加 1，然后再使用**自增后的这个变量**的值。
```cpp
#include <iostream>

int main()
{
    int a = 5;
    std::cout << "前置自增前：a = " << a << std::endl;      // 输出：5
    std::cout << "前置自增：++a = " << ++a << std::endl;    // 输出：6
    std::cout << "前置自增后：a = " << a << std::endl;      // 输出：6
    return 0;
}
```

编译运行这段代码，会输出：
```
前置自增前：a = 5
前置自增：++a = 6
前置自增后：a = 6
```

## 1.2 后置自增

后置自增的语法是 `变量名++`，它会先使用**自增前这个变量**的值，然后再将变量的值增加 1。
```cpp
#include <iostream>

int main()
{
    int a = 5;
    std::cout << "后置自增前：a = " << a << std::endl;      // 输出：5
    std::cout << "后置自增：a++ = " << a++ << std::endl;    // 输出：5
    std::cout << "后置自增后：a = " << a << std::endl;      // 输出：6
    return 0;
}
```

编译运行这段代码，会输出：
```
后置自增前：a = 5
后置自增：a++ = 5
后置自增后：a = 6
```

# 2. 自减运算符

自减运算符 `--` 与自增运算符类似，它的作用是将一个变量的值减少 1。
自减运算符也有两种使用方式：前置自减和后置自减，同样地：
 - 前置自减会先将变量的值减少 1，然后再使用**自减后的这个变量**的值。
 - 后置自减会先使用**自减前这个变量**的值，然后再将变量的值减少 1。

```cpp
#include <iostream>

int main()
{
    int a = 5;
    std::cout << "前置自减前：a = " << a << std::endl;      // 输出：5
    std::cout << "前置自减：--a = " << --a << std::endl;    // 输出：4
    std::cout << "前置自减后：a = " << a << std::endl;      // 输出：4

    a = 5; // 重置 a 的值
    std::cout << "后置自减前：a = " << a << std::endl;      // 输出：5
    std::cout << "后置自减：a-- = " << a-- << std::endl;    // 输出：5
    std::cout << "后置自减后：a = " << a << std::endl;      // 输出：4
    return 0;
}
```

编译运行这段代码，会输出：
```
前置自减前：a = 5
前置自减：--a = 4
前置自减后：a = 4
后置自减前：a = 5
后置自减：a-- = 5
后置自减后：a = 4
```
