# 1. 变量

豆包给出的变量的定义：

![[1.png]]


可以用以下代码来声明变量：
```cpp
int var_int = 10;               // 声明一个整数变量 var_int，并初始化为 10
double var_double = 3.14;       // 声明一个双精度浮点变量 var_double，并初始化为 3.14
char var_char = 'A';            // 声明一个字符变量 var_char，并初始化为 'A'
bool var_bool = true;           // 声明一个布尔变量 var_bool，并初始化为 true
```

注：
1. C++ 中的等号是赋值运算符，用于将右侧的值赋给左侧的变量。
2. 变量可以在声明时初始化，也可以在之后的代码中赋值。
```cpp
int a = 10;                     // 声明并初始化变量 a
int b;                          // 声明变量 b
b = 20;                         // 在声明后再给变量 b 赋值
```

# 2. int 类型变量的内存布局

在 C++ 中，int 类型通常占用 4 个字节（32 位），以 10 为例，10 的二进制是 $1010_2$：

```
int 类型变量 var_int 占用的内存空间：
0000 0000 | 0000 0000 | 0000 0000 | 0000 1010
```

这里的最高位（最左边的位）是符号位，0 表示正数，1 表示负数。对于正数，剩余的位表示数值；对于负数，使用补码表示。
比如 -10：

```
int 类型变量 var_int 占用的内存空间：
1111 1111 | 1111 1111 | 1111 1111 | 1111 0110
```

因此，int 类型变量一共可以表示 $2^{32}$ 个不同的值，其中一半是正数（包括 0），另一半是负数。
正数的范围是 1 到 $2^{31}-1$，负数的范围是 -1 到 $-2^{31}$，还有一个 0。

---------------------------------------------------------------------
$$2^{32} = 4294967296$$
$$2^{31} - 1 = 2147483647$$
$$-2^{31} = -2147483648$$
---------------------------------------------------------------------

# 3. 常见的变量占据的字节数

| 数据类型 | 内存空间      |
| -------- | ------------- |
| int      | 4字节（32位） |
| double   | 8字节（64位） |
| float    | 4字节（32位） |
| char     | 1字节（8位）  |
| bool     | 1字节（8位）  |

# 4. 整数类型的变量

常见的整数类型变量有 `int`、`short`、`long`、`long long` 以及它们的无符号版本 `unsigned int`、`unsigned short`、`unsigned long`、`unsigned long long`。

| 数据类型            | 32位系统（x86）  | 64位系统（x64）  |
| --------------- | ----------- | ----------- |
| bool            | 1字节（8位）     | 1字节（8位）     |
| short/short int | 2字节（16位）    | 2字节（16位）    |
| long/long int   | 4字节（32位）    | 8字节（64位）    |
| long long       | 8字节（64位）    | 8字节（64位）    |
| unsigned 类型     | 和对应的有符号类型相同 | 和对应的有符号类型相同 |

值得注意的是，`int` 在 32 位系统中占用 4 字节，在 64 位系统中也占用 4 字节，而 `long` 在 32 位系统中占用 4 字节，在 64 位系统中占用 8 字节。这是因为 C++ 标准并没有规定具体的数据类型大小，而是由编译器和平台决定的。
