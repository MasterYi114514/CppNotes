# 1. 常量

常量（constant）是指在程序运行过程中其值不能改变的量。常量可以是数字、字符、字符串等类型，在程序中通常用来表示一些固定的值，如数学常数、物理常数等。

在 C++ 中，可以使用 `const` 关键字来定义常量。
```cpp
const double pi = 3.14159;
```

在上面的代码中，`pi` 是一个常量，值为 3.14159，这里的 `const` 关键字也可以在类型的后面，即 `double const pi = 3.14159;`，两者是等价的。

注：
**1. 常量必须在定义时初始化。**

对于以下代码，声明常量的时候没有初始化：
```cpp
#include <iostream>

int main()
{
    const int a;

    return 0;
}
```

编译该程序，得到以下报错：
```
code2.cpp: In function 'int main()':
code2.cpp:5:15: error: uninitialized 'const a' [-fpermissive]
    5 |     const int a;
      |               ^
```

**2. 常量的值在程序运行过程中不能被修改，如果尝试修改常量的值，编译器会报错。**

对于以下代码，尝试修改常量的值：
```cpp
#include <iostream>

int main()
{
    const int b = 10;
    b = 20;

    return 0;
}
```

编译该程序，得到以下报错：
```
code3.cpp: In function 'int main()':
code3.cpp:6:7: error: assignment of read-only variable 'b'
    6 |     b = 20;
      |     ~~^~~~
```

这里对常量的称呼 `read-only variable` 翻译过来是“只读变量”，也就是说常量的值只可以被读取，不能被修改。

# 2. 字面常量

字面常量（literal constant）是指在程序中直接使用的固定值，如数字、字符、字符串等。这些值在程序运行过程中不会改变。
```cpp
#include <iostream>

int main()
{
    int a = 100;                // 100 是一个字面常量
    double b = 3.14;            // 3.14 是一个字面常量
    char c = 'A';               // 'A' 是一个字面常量

    // "Hello world" 是一个字符串字面常量
    std::cout << "Hello world" << std::endl;

    bool real = true;           // true 是一个字面常量
    bool fake = false;          // false 是一个字面常量

    return 0;                   // 0 是一个字面常量
}
```
