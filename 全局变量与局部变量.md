# 1. 变量的作用域

学习代码块的时候，我们指出，C++ 中依靠代码块来划分变量的作用域。

每队 `{}` 括起来的代码块都可以定义一个作用域，在这个作用域内定义的变量只能在这个作用域内访问，出了这个作用域就无法访问了。
```cpp
#include <iostream>

int main()
{ // 1. main 函数域开始
    int a = 10;         // 变量 a 的作用域是 1 号 main 函数域

    // 这里是 1 号 main 函数域可以访问变量 a
    std::cout << "变量 a 的值：" << a << std::endl;     

    { // 2. 内部代码块域开始，内部代码块是 main 函数域的一部分
        int b = 20;     // 变量 b 的作用域是 2 号内部代码块域

        std::cout << "变量 b 的值：" << b << std::endl;

        // 因为这里是 1 号 main 函数域的一部分，所以这里也可以访问变量 a
        std::cout << "变量 a 的值：" << a << std::endl;

        { // 3. 在里面再嵌套一层 3 号内部代码块
            int c = 30; // 变量 c 的作用域是这个最内层的代码块域

            std::cout << "变量 c 的值：" << c << std::endl;

            // 因为这里是 main 函数域的一部分，所以这里也可以访问变量 a
            std::cout << "变量 a 的值：" << a << std::endl;

            // 因为这里是 2 号内部代码块域的一部分，所以这里也可以访问变量 b
            std::cout << "变量 b 的值：" << b << std::endl;
        }

        std::cout << c << std::endl;        // 错误，超出 3 号内部代码块域，无法访问变量 c
    } // 2. 内部代码块域结束 

    std::cout << b << std::endl;            // 错误，超出 2 号内部代码块域，无法访问变量 b
} // 1. main 函数域结束
```

注：
1. 由上面的例子可以看出，代码块之间出现嵌套时，**内层代码块可以访问外层代码块中定义的变量**，但外层代码块无法访问内层代码块中定义的变量。
2. 在嵌套的代码块中，如果内层代码块定义了一个与外层代码块同名的变量，那么内层代码块中的变量会隐藏外层代码块中的同名变量。
```cpp
#include <iostream>

int main()
{
    int a = 10, b = 20;         // 外层代码块中的变量 a 和 b

    { // 内层代码块开始
        int a;                  // 在内层代码块中定义了另一个变量，名字也叫 a

        a = 100;        // 因为名字一样，编译器优先找到内层的 a 进行修改
        b = 50;         // 这里是内层可以访问外层的变量 b，所以修改了外层的 b 的值

        // 输出内层代码块中的变量 a 和 b 的值
        std::cout << "a 和 b 的值：" << a << " " << b << std::endl; 
    } // 内层代码块结束

    // 输出外层代码块中的变量 a 和 b 的值
    std::cout << "a 和 b 的值：" << a << " " << b << std::endl;

    return 0;
}
```

编译并运行该代码后，得到以下输出：
```
a 和 b 的值：100 50
a 和 b 的值：10 50
```

# 2. 全局变量与局部变量

`{}` 所划定的代码块都是局部的作用域，在这个作用域内定义的变量叫做局部变量，局部变量只能在这个作用域内访问。

而在所有的 `{}` 之外定义的变量叫做全局变量，全局就是所有局部代码块的外层代码块，全局变量可以在任何地方访问。

```cpp
#include <iostream>

int global_var = 10;        // 全局变量

int main()
{
    // 可以访问全局变量
    std::cout << "全局变量 global_var 的值：" << global_var << std::endl; 

    int local_var = 20;      // 局部变量

    // 可以访问局部变量
    std::cout << "局部变量 local_var 的值：" << local_var << std::endl; 

    return 0;
}
```
