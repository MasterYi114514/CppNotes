# 1. 基础语句

语句是 C++ 中的最小执行单元，以分号 `;` 结尾。每条语句执行一个特定的操作，例如声明变量、赋值、调用函数等。

```cpp
int a = 10;         // 一个声明变量并赋值的语句
std::cout << a;     // 一个调用函数的语句
```

# 2. 复合语句

复合语句是由一对大括号 `{}` 包围的一组语句，表示一个代码块。复合语句可以包含多条语句，并且在控制结构（如条件语句、循环语句）中经常使用。

```cpp
#include <iostream>

int main()
{
    {
        int a = 10;
        double pi = 3.14159;
        std::cout << "a: " << a << std::endl;       // 输出变量 a 的值
        std::cout << "pi: " << pi << std::endl;     // 输出变量 pi 的值
    }

    std::cout << "变量 a 和 pi 已经超出作用域，无法访问它们了。" << std::endl;
    // std::cout << "a: " << a << std::endl;

    return 0;
}
```

注：
1. 在 C++ 中，变量的作用域是由代码块决定的。这里的“作用域”是指变量在程序中可以被访问的范围，超出作用域的变量就会被销毁，无法再访问。
2. 在下面的例子中，全局变量 `global_var` 可以在任何地方访问，而局部变量 `a` 和 `b` 只能在它们所在的代码块内访问。
```cpp
#include <iostream>

int global_var = 42;            // 全局变量

int main()
{
    std::cout << "全局变量可以在任何地方访问: " << global_var << std::endl;

    int a = 10;                 // main 函数内的局部变量
    std::cout << "局部变量 a 的值: " << a << std::endl;

    {
        std::cout << "该代码块在 main 函数的代码块内部。" << std::endl;
        std::cout << "可以在这里访问变量 a: " << a << std::endl;
        std::cout << "当然也可以访问全局变量: " << global_var << std::endl;

        int b = 20;             // 内部代码块的局部变量
        std::cout << "内部代码块的局部变量 b 的值: " << b << std::endl;
    }

    std::cout << "这里就超出了内部代码块的作用域，无法访问变量 b 了。" << std::endl;
    // std::cout << "变量 b 的值: " << b << std::endl;

    return 0;
}
```
