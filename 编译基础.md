
# 1. 可执行文件

**可执行文件（executable file）**：能被操作系统直接加载，让计算机执行的文件。

在 Windows 上，常见的可执行文件扩展名是 `.exe`，在 Linux 和 macOS 上则没有特定的扩展名，有时会使用 `.out`。

# 2. 编译

**编译**：将源代码转换成可执行文件的过程。

C++文件的编译过程可以简单地分为以下几个步骤：
```
a.cpp(源代码) -> a.i(预处理后文件) -> a.s(汇编代码) -> a.o(目标文件) -> a.exe(可执行文件)
```

# 3. 编辑器、编译器和IDE

**编辑器（editor）**：用于编写代码的软件，如 VisualStudioCode（VScode）等。
**编译器（compiler）**：将源代码转换成可执行文件的程序，如 g++、clang++ 等。
**集成开发环境（IDE）**：集成了编辑器、编译器和调试工具的开发环境，如 Visual Studio、Dev-C++、Code::Blocks 等。
