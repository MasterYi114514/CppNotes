# 1. 赋值语句

**赋值语句**：用于将一个值赋给一个变量的语句。

我们用单个的 `=` 表示赋值的含义：
```cpp
int a;          // 定义一个整数类型的变量 a
a = 5;          // 将值 5 赋给变量 a

int b = 10;     // 定义一个整数类型的变量 b，并初始化为 10
b = b + 3;      // 先计算 b + 3 的结果，然后将结果赋值给变量 b
```

# 2. 复合赋值运算符

**复合赋值运算符**：是一种简化的赋值语句，可以将一个变量与一个表达式进行运算，并将结果赋值给该变量。
```cpp
#include <iostream>

int main()
{
    int a = 10, b = 5;

    a += 5;                             // 等同于 a = a + 5
    std::cout << a << std::endl;        // 输出 15
    a += b;                             // 等同于 a = a + b
    std::cout << a << std::endl;        // 输出 20
    a *= 2;                             // 等同于 a = a * 2
    std::cout << a << std::endl;        // 输出 40
    a /= 4;                             // 等同于 a = a / 4
    std::cout << a << std::endl;        // 输出 10
    a -= 3;                             // 等同于 a = a - 3
    std::cout << a << std::endl;        // 输出 7
    a %= b;                             // 等同于 a = a % b
    std::cout << a << std::endl;        // 输出 2

    return 0;
}
```

# 3. 赋值语句的返回值

赋值语句本身也是一个表达式，它的值是**被赋值后的变量的值**。

换言之，对于 (a = 5) 这个赋值语句来说，它把 5 赋值给了变量 a，同时这个赋值语句的值是**赋值后的a的值**。

当然，我这里并没有直接说它的值是5，因为我们目前遇到的变量类型都很简单，以后遇到很复杂的对象类型时，这个语句的返回值可能需要根据对象的设计来仔细推导。
```cpp
#include <iostream>

int main()
{
    int a = 10;
    std::cout << (a = a + 100) << std::endl;        // 输出 110，此时 a 的值已经被修改为 110
    std::cout << a << std::endl;                    // 输出 110

    return 0;
}
```

注：
1. 这个值是被赋值后的变量的值，所以我们也可以在一个表达式中连续使用多个赋值语句：
```cpp
#include <iostream>

int main()
{
    int a = 10, b = 20, c = 30;
    std::cout << (a = b = c) << std::endl;                      // 输出 30，此时 a、b、c 的值都被修改为 30
    std::cout << a << " " << b << " " << c << std::endl;        // 输出 30 30 30

    return 0;
}
```

2. 赋值语句也可以用于 if 语句等进行真假判断的地方，注意不要与比较运算符 `==` 混淆了：
```cpp
#include <iostream>

int main()
{
    int a = 99;

    if(a = 100)         // 错误的使用了赋值运算符 =，而不是比较运算符 ==
    {
        std::cout << "这个变量 a 竟然恰好是 100 诶！" << std::endl;
    }
    else
    {
        std::cout << "我们没有这么幸运, a 的值不是 100, 而是 " << a << std::endl;
    }

    return 0;
}
```

这段代码中，a 的值并不是 100，但是我们进行了错误判断，运行后会输出：
```
这个变量 a 竟然恰好是 100 诶！
```
