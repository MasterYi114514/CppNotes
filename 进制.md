
# 1. 进制

我们日常用的数字是十进制，比如114514，可以进行这样的拆分：

$$114514 = 1 * 10^5 + 1 * 10^4 + 4 * 10^3 + 5 * 10^2 + 1 * 10^1 + 4 * 10^0$$

这里的 10 就是进制的基数，$10^n$ 就是权重。

可以类似地，给出 R 进制的定义：

---------------------------------------------------------------------


在基数为 R 的进制中,包含 R 个数字符号, 分别是 0, 1, 2, ..., R-1
一个整数n位、小数m位的R进制数字，可以表示为：

$$(N)_{R} = (K_{n-1} K_{n-2} ... K_0.K_{-1}K_{-2}...K_{-m})_{R}$$

其中，$K_i$ 是第 $i$ 位上的数字符号，满足 $0 \leq K_i < R$。
同时，也可以用连加符号来表示：

$$(N)_{R} = \sum_{i=-m}^{n-1} K_i R^i$$

---------------------------------------------------------------------

# 2. 二进制

二进制是基数为 2 的进制系统，使用两个数字符号：0 和 1。每一位上的权重是 2 的幂次方。

例如：

---------------------------------------------------------------------

$$1101_2 = 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0 = 8 + 4 + 0 + 1 = 13_{10}$$
