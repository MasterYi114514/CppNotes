# 1. 数组

**数组（Array）**：同一类型数据的连续内存集合。

数组的所有元素类型都是相同的，并且长度是固定的，C++ 中可以用 `[]` 来定义数组。
```cpp
// 类型 数组名[数组长度];
int arr[5];             // 定义一个长度为 5 的 int 类型的整数数组
char string[100];       // 定义一个长度为 100 的字符数组（字符串）
```

注：
标准 C++ 要求定义数组时，数组的长度必须是一个编译时就可以确定的常量表达式，不能是一个变量或者运行时才能确定的值。
```cpp
#include <iostream>

int main()
{
    const int N = 5;        // N 是编译时就能确定的 const 常量
    constexpr int M = 10;   // M 是编译时就能确定的 constexpr 常量

    int x; std::cin >> x;        // x 是一个变量，运行时才能确定其值
    const int y = x;        // y 是运行时才能确定的 const 常量

    int arr[5];             // 正确，5 是字面量 
    int arr1[N];            // 正确，N 是编译时常量
    int arr2[M];            // 正确，M 是编译时常量
    int arr3[x];            // 错误，x 是变量
    int arr4[y];            // 错误，y 是运行时常量

    return 0;
}
```

但是，GCC 和 Clang 等编译器支持一种称为变长数组（Variable Length Array，VLA）的扩展特性，允许在定义数组时使用一个运行时才能确定的变量作为数组长度，因此直接编译运行上面的代码也是可以的。

我们在编译指令中加上 `-pedantic-errors` 选项来启用严格的标准兼容性检查，再进行编译，就会得到报错：
```
code1.cpp: In function 'int main()':
code1.cpp:14:9: error: ISO C++ forbids variable length array 'arr3' [-Wvla]
   14 |     int arr3[x];            // 错误，x 是变量
      |         ^~~~
code1.cpp:15:9: error: ISO C++ forbids variable length array 'arr4' [-Wvla]
   15 |     int arr4[y];            // 错误，y 是运行时常量
      |         ^~~~
```

## 1.1 通过 `[]` 访问数组元素

数组元素的访问通过 `[]` 来实现，数组的索引从 0 开始。
```cpp
#include <iostream>

int main()
{
    int arr[5];

    arr[0] = 10;        // 给数组的第一个元素赋值
    arr[4] = 50;        // 给数组的最后一个元素赋值
    arr[5] = 100;       // 错误，arr 的长度为 5，下标范围是 [0,4]
    return 0;
}
```

## 1.2 对数组进行初始化

数组可以在定义时进行初始化，使用大括号 `{}` 来指定数组元素的初始值。

**1. 完全显式初始化**

用 `{}` 来依次排列出要初始化的元素值，中间用逗号分隔。
```cpp
int arr[5] = {1, 2, 3, 4, 5};       // 定义并初始化一个长度为 5 的整数数组
```

**2. 部分初始化**

`{}` 中列出的元素值的个数小于数组长度，未列出的元素会被自动初始化为 0。
```cpp
int arr1[5] = {1, 2};
// 将 arr[0] 初始化为 1
// 将 arr[1] 初始化为 2
// arr[2]、arr[3]、arr[4] 被初始化为 0

int arr2[5] = {};       // 空大括号会自动将 arr2 中的所有元素初始化为 0
```

**3. 不指定数组长度，由编译器进行推导**

在定义数组时，`[]` 中不指定数组长度，编译器会根据 `{}` 中列出的元素值的个数来推导数组的长度。
```cpp
int arr[] = {1, 2, 3, 4, 5};       // 定义并初始化一个整数数组，长度由编译器推导为 5
```

更详细的内容，可以参考[[初始化列表]]

## 1.3 数组的内存布局

此小节需要预先学习：[[比特、字节、内存与地址#^195831]]

数组在内存中是连续存储的，每个元素占用相同大小的内存空间。

比如对于 `int arr[5]`，数组的每个元素都占用四个字节的空间，我们用其中第一个字节的地址来表示整个元素的地址。假设 `arr[0]` 的地址是 `0x12345678ABCD1234`，那么数组的内存布局如下：
```
arr[0] --- 0x12345678ABCD1234
arr[1] --- 0x12345678ABCD1238
arr[2] --- 0x12345678ABCD123C
arr[3] --- 0x12345678ABCD1240
arr[4] --- 0x12345678ABCD1244
```

# 2. 常量数组

我们可以在定义数组时使用 `const` 关键字来修饰数组元素，使得数组元素的值不能被修改。
```cpp
#include <iostream>

int main()
{
    const int arr[5] = {1, 2, 3, 4, 5};     // 定义一个 const 数组
    const char str[] = "Hello";             // 定义一个 const 字符数组（字符串）

    arr[0] = 10;        // 错误，不能修改 const 数组的元素

    return 0;
}
```

注：
`const int arr[]` 在这里的含义是 `arr` 数组中的每一个元素都是 `const int` 类型的，不可以被修改。因此，对于用 `const` 修饰的常量数组，和常量一样，**必须在定义的时候进行初始化**，并且**不能在后续的代码中修改数组元素的值**。
