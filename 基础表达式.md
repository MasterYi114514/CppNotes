# 1. 表达式

**表达式**：由一个或多个常量、变量、函数调用、运算符等组成的一行代码，能够计算出一个值。

表达式可以是一个简单的常量或变量，也可以是一个复杂的函数调用或运算组合。

简言之，表达式是一个**能够被计算出结果**的代码片段。

# 2. 算数表达式

**算数表达式**：由算数运算符（如 `+`、`-`、`*`、`/` 等）连接的表达式，用于进行数学计算。
```cpp
a + b           // 加法表达式
a - b           // 减法表达式
a * b           // 乘法表达式
a / b           // 除法表达式
a % b           // 取模表达式
(a + b) * c     // 复合算数表达式
```

注：
1. 取模运算：
```
对于任意两个整数 被除数a 和除数b（b不为0）
a % b 的结果是 a 除以 b 的余数，也记作 a mod b
满足恒等式：
$$ a = b \times q + r $$
其中 q 是 a 除以 b 的商（向零取整），r 是 a 除以 b 的余数
有以下特点：
    1. 余数 r 的绝对值总是小于除数 b 的绝对值
    也就是说，满足以下不等式：
    $$ 0 \leq |r| < |b| $$

    2. 余数 r 的符号与被除数 a 的符号相同

e.g. 7 mod 3 = 1, -7 mod 3 = -1, 7 mod -3 = 1, -7 mod -3 = -1
```

2. 除法运算：

这里的 “/” 运算符表示整数除法，它的结果会根据参与运算的数据类型的不同而发生改变：
- 当两个操作数都是整数类型时，结果也是整数类型，且会向零取整。
- 当至少一个操作数是浮点类型时，结果也是浮点类型，且会进行正常的除法计算。

给出以下示例代码：
```cpp
#include <iostream>

int main()
{
    int a_int = 5, b_int = 2;
    double a_double = 5, b_double = 2;

    std::cout << "int / int       : " << a_int / b_int << std::endl;
    std::cout << "double / int    : " << a_double / b_int << std::endl;
    std::cout << "int / double    : " << a_int / b_double << std::endl;
    std::cout << "double / double : " << a_double / b_double << std::endl;

    return 0;
}
```

编译运行后得到输出：
```
int / int       : 2
double / int    : 2.5
int / double    : 2.5
double / double : 2.5
```

# 3. 关系表达式

该小节需要预先学习：[[if 语句]]

**关系表达式**：由关系运算符（如 `==`、`!=`、`<`、`>`、`<=`、`>=` 等）连接的表达式，用于比较两个值的大小或相等性。

```cpp
a > b
a < b
a >= b
a <= b
a == b          // 判断二者相等，注意是两个等号
a != b          // 判断二者不等
```

注：
1. 关系表达式的值是布尔类型（true 或 false），当两个操作数满足该关系时，表达式的结果为 true；否则为 false。
2. 在 C++ 中，整数的数值也可以当做布尔类型来用，非零值表示 true，零值表示 false。

参考这段代码：
```cpp
#include <iostream>

int main()
{
    if(1) std::cout << "1 被认为是 true" << std::endl;

    if(0) std::cout << "这句话不会被输出" << std::endl;
    else std::cout << "0 被认为是 false" << std::endl;

    if(-1) std::cout << "-1 也被认为是 true" << std::endl;
    if(100) std::cout << "100 也被认为是 true" << std::endl;

    int a = 2;
    if(a) std::cout << "也可以用变量的值来判断条件" << std::endl;

    return 0;
}
```

编译运行后，得到如下输出：
```
1 被认为是 true
0 被认为是 false
-1 也被认为是 true
100 也被认为是 true
也可以用变量的值来判断条件
```
